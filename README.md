# Restaurant Platform

Monorepo scaffold for restaurant customer apps, dashboards, backend services, and shared packages.

## Workspace structure

- `apps/` – frontend and backend applications
- `packages/` – shared libraries and contracts
- `infra/` – deployment infrastructure
- `docs/` – documentation and diagrams

## Getting started

1. Install dependencies:

```bash
pnpm install
```

2. Run a task (example):

```bash
tpn run build
```
